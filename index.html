Java中注解的实现

注解简介

从Java源码可以看到， 注解的本质是一个接口，

注解携带的是元数据，但不会对被注释的代码逻辑产生影响

元注解

元注解即注解的注解，java中的元注解有：

  • @Target
  • @Retention
  • @Document
  • @Inherited

元注解只在定义注解时使用

@Target

指定被修饰注解的作用对象

可以看到value是一个数组，即注解的作用对象可以是多个，取值范围都在ElementType

这个枚举类

@Retention

指定被修饰的注解的生命周期

可以看到Retention注解带有一个RetentionPolicy的枚举值

@Documented

表示是否添加到 java doc中

@Inherited

表示注解是否被继承
